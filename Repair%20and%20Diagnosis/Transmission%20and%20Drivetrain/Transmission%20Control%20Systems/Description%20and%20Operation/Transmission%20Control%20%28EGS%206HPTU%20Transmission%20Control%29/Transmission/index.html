Transmission




General Reference To SBT

Functional description, see BMW Technology:
TIS -> Document -> SI Technology -> SBT Number
- 2401 03 040

The automatic transmissions 6HP19, 6HP26 and 6HP32 are mechatronic systems that unite both the mechanical system (gearbox) and the electronics (transmission control) within the gearbox case. In the same way as automatic transmissions used to date, the adaptive transmission control adapts the shift characteristics of the comfort program and sport program to the driver's demand and the driving situation.
