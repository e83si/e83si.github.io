Electronics Module




Mechatronic of the Automatic Transmission

The Mechatronics of the 6-HP automatic transmission family consists essentially of the electronic module and the hydraulic module.

More detailed functional description, see BMW Technology:
TIS -> Document -> SI Technology -> SBT Number
- 2401 03 040

Electronic module
The electronic module is divided mainly into the following components:
- EGS control module
- Power supply for solenoid valves and electronic pressure control valves
- Transmission sensor system:
- Sensor for the transmission input speed
- Sensor for the transmission output speed
- Transmission oil temperature sensor

- Transmission connector for connection to vehicle wiring harness

EGS control module
The EGS on the-HP transmission is fitted as part of the transmission and no longer as a separate control module in the E box. It evaluates the sensors electronically and directly activates the actuators of the hydraulic module. A temperature sensor of the EGS processor additionally monitors the transmission oil temperature.

Hydraulic module
The hydraulic module is similar to the modules used so far in transmissions. It is also designed as holder for the electronic module.
