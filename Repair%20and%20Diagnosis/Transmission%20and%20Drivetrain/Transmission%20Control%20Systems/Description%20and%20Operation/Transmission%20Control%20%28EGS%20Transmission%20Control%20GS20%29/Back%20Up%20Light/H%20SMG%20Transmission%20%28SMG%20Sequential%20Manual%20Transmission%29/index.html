H SMG Transmission (SMG Sequential Manual Transmission)



H SMG Transmission

Functional description see BMW Technology: TIS -> Document -> SI Technology - SBT Numbers: - for E46: 2301 03 002 - for E85: 2302 03 021

Sequential Manual Transmission SMG means that in contrast to conventional manual transmissions the clutch control and gear changes are automated. An electrohydraulic control system handles the functions that are otherwise performed by the driver shifting with the clutch pedal, gear lever and accelerator pedal.
