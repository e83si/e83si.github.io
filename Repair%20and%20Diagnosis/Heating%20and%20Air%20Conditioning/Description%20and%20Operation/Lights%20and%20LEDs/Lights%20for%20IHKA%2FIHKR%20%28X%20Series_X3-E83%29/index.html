Lights for IHKA/IHKR (X Series_X3-E83)




Lights For IHKA/IHKR

The function and indicator lighting is dimmed as follows:
- Dependent on the position of the dimmer control on the light switch center
- Dependent on the ambient brightness (photocell on light switch center)

Function indicator lamps
The following buttons feature a green function light (LED):
- IHKA: AUTO, air conditioner and keys for individual air distribution. The button for AUC/recirculating air has 2 green function lights (LEDs).
- IHKR: recirculated air, air conditioner

The buttons for the rear window defogger and defrost button feature an orange-colored function light (LED) on both model versions.

Locating light
When the light is switched on, the indicator light is activated via terminal 58g. The signal comes from the light switch center (LSZ).

NOTE: If the locating light does not function, check the connection to the light switch center. If this is OK and the locating light of other components is OK, replace the control module/operating unit of the heating/air conditioning.
