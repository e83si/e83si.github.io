Heater Blower With Blower Rocker Switch +/- (IHKA Integrated Heating/Air Conditioning Control E83)



Heater Blower With Blower Rocker Switch +/-

Heater blower with blower rocker switch +/-
The blower capacity can be set in stages with a rocker switch. This is indicated by a blower bar decreasing in the display.

The blower motor is controlled by a linear blower output stage. Depending on the set stage the blower voltage ranges from 0 Volt to maximum 8 Volt. The blower is switched off at a blower control voltage below 0.5 Volt.

In the automatic program, the set stage is influenced depending on the temperature difference between the set point and the interior temperature. The greater the difference, the higher the increase in the set stage. In order to avoid cold air being blown in, the blower is started up in heating mode dependent on the heat exchanger temperature.
