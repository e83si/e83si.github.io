Electrical - Oil Level Sensor Diagnostic Hints

SI B 12 33 06
Engine Electrical Systems

October 2006
Technical Service

This Service Information bulletin supersedes S.M. B12 217 04 dated December 2004.

SUBJECT
Various Complaints caused by Defective Oil Level Sensor - Diagnostic Hints

MODEL
E46 M54 (3 Series)
E53 (X5) All Models
E60 N62 (5 Series)
E63, E64 (6 Series)
E65, E66 N62 (7 Series)
E83 (X3) All Models
E85 (Z4) All Models

SITUATION

E46 M54 equipped vehicles:
Customer complains the vehicle does not crank and when diagnosed using the DIS Plus or GT1, FC 060 (3C) Selector Switch, is stored in the EGS Control Module.

E53 M54 equipped vehicles:
Customer complains vehicle does not crank, vehicle charging system is inoperative, engine temperature gauge is inoperative, the universal garage door opener is inoperative and when diagnosed using the DIS Plus or GT1 FC 060 (3C) Selector Switch is stored in the EGS Control Module.

E53 M62 equipped vehicles:
Customer complains the vehicle charging system is inoperative.

E53 N62 equipped vehicles:
Customer complains the vehicle charging system is inoperative, check engine lamp is illuminated and when diagnosed using the DIS Plus or GT1 the DME stores various oxygen sensor faults.

E60, E63, E64, E65 and E66 N62 equipped vehicles:
Customer complains the Control Display intermittently indicates "Engine Temperature Fault" with no other driveability or overheating complaints. When diagnosed using the DIS Plus or GT1, the DME stores various oxygen sensor faults.

E83 M54 equipped vehicles:
Customer complains vehicle does not crank and when diagnosed using the DIS Plus or GT1 FC 060 (3C) Selector Switch is stored in the EGS Control Module.

E85 M54 equipped vehicles:
Customer complains vehicle does not crank and when diagnosed using the DIS Plus or GT1, FC 060 (3C) Selector Switch is stored in the EGS Control Module.

CAUSE

E46 M54 equipped vehicles:
Defective Oil Level Sensor causing a loss of voltage to all components powered by Fuse 30.

E53 M54 equipped vehicles:
Defective Oil Level Sensor causing a loss of voltage to all components powered by Fuse 5.

E53 M62 equipped vehicles:
Defective Oil Level Sensor causing a loss of voltage to all components powered by Fuse 5.

E53 N62 equipped vehicles:
Defective Oil Level Sensor causing a loss of voltage to all components powered by Fuse 008 located in the IVM.

E60, E63, E64, E65 and E66 N62 equipped vehicles:
Defective Oil Condition Sensor causing a loss of voltage to all components powered by Fuse 008 located in the IVM.

E83 M54 equipped vehicles:
Defective Oil Level Sensor causing a loss of voltage to all components powered by Fuse 30.

E83 M54 equipped vehicles:
Defective Oil Level Sensor causing a loss of voltage to all components powered by Fuse 9.

PROCEDURE

E46 M54 equipped vehicles:
Check fuse 30. If found defective, replace the fuse and Oil Level Sensor.

If fuse 30 is not defective, check for a voltage drop across fuse, if a voltage drop is present remove connector X6254 from Oil Level Sensor. If the voltage drop is no longer present, replace the Oil Level Sensor.

E53 M54 and M62 equipped vehicles:
Check fuse 5. If found defective, replace the fuse and Oil Level Sensor.

If fuse 5 is not defective check for a voltage drop across fuse, if a voltage drop is present remove connector X6254 from Oil Level Sensor. If the voltage drop is no longer present, replace Oil Level Sensor.

E53 N62 equipped vehicles:
1. Check IVM fuse 008. If found defective, replace the fuse and Oil Level Sensor.
2. If IVM fuse 008 is not defective check for voltage drop across fuse, if a voltage drop is present remove connector X6254 from Oil Level Sensor. If the voltage drop is not longer present, replace the Oil Level Sensor.

E60, E63, E64, E65 and E66 N62 equipped vehicles
1. Check IVM fuse 008. If found defective, replace the fuse and Oil Level Sensor.
2. If IVM fuse 008 is not defective check for a voltage drop across the fuse, if a voltage drop is present remove connector X6254 from Oil Level Sensor. If the voltage drop is no longer present, replace the Oil Level Sensor.

E83 M54 equipped vehicles:
1. Check fuse 30. If found defective, replace the fuse and Oil Level Sensor.
2. If fuse 30 is not defective check for a voltage drop across fuse, if a voltage drop is present, remove connector X6254 from Oil Level Sensor. If the voltage drop is no longer present replace the Oil Level Sensor.

E85 M54 equipped vehicles
1. Check fuse 9. If found defective, replace the fuse and Oil Level Sensor.
2. If fuse 9 is not defective check for a voltage drop across fuse, if a voltage drop is present, remove connector X6254 from Oil Level Sensor. If the voltage drop is no longer present replace the Oil Level Sensor.
