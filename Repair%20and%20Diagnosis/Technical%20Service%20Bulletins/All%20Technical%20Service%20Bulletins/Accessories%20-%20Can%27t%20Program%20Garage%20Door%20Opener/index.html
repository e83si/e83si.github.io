Accessories - Can't Program Garage Door Opener

SI B66 06 07
Distance Systems, Cruise Control, Remote Control

October 2007
Technical Service

SUBJECT
Universal Garage Door Opener Cannot be Programmed

MODEL

E60, E61 (5 Series)

E63, E64 (6 Series)

E65, E66 (7 Series)

E70 (X5)

E83 (X3)

E90, E91, E92, E93 (3 Series)

SITUATION
The universal garage door opener (UGDO), option code 319, cannot be programmed and/or taught-in and as such appears not to be working properly.

CAUSE

1. The correct procedure for programming the UGDO is not being followed.

2. The batteries in the customer's transmitter are not sufficiently charged.

CORRECTION

1. Ensure that the programming procedure is being performed correctly and is understood by the customer.

2. Attempt to program the UGDO again using new batteries in the transmitter.

PROCEDURE

1. Ensure that the customer has attempted the programming procedure correctly according to the instructions in the owner's manual.

- The procedure in the owner's manual can be found in the "Controls" section of the manual.

- NOTE: Additional information about the HomeLink UGDO can be found at www.homelink.com/home/home.tml

2. Using new batteries in the transmitter, attempt to program the UGDO.

3. NOTE: If the customer has both a regular, full size transmitter as well as a smaller (key ring) transmitter, only the full size transmitter should be used for the programming procedure.

WARRANTY INFORMATION

Covered under the terms of the BMW New Vehicle Limited Warranty.

Please refer to the latest KSD for all applicable labor operations and allowances.

If the appropriate labor operation is not contained in KSD, then a work time labor operation should be used.

Warranty claims submitted to replace the UGDO for the above mentioned reason should not be made and will not be reimbursed.
