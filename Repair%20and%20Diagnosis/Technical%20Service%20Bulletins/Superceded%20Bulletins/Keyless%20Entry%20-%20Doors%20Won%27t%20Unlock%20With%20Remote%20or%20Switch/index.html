Keyless Entry - Doors Won't Unlock With Remote or Switch

SI B51 20 08
Body Equipment

July 2008
Technical Service

SUBJECT
Doors Will Not Unlock with the Remote or Central Locking Switch

MODEL
E83 (X3) equipped with the Ambient Light Package (SA 563) and produced from 9/2007

SITUATION
Intermittently, the doors will not unlock when pressing the unlock button on the remote, or when pressing the central locking switch.

In most cases, the vehicle can be unlocked by pressing the unlock button a second time.

If the unlock function is still not restored immediately, the vehicle can still be entered by using the key in the lock cylinder, or exited by pulling the interior door handle twice.

No relevant fault codes are stored in the General Module (GM5).

CAUSE
Software error in the GM5

PROCEDURE

^ Do not replace any parts or program and/or code the vehicle, as these will not provide a solution.

^ Please explain to the customer that this intermittent problem is known, and that a software solution will be available shortly.

^ As soon as a solution is available, this Service Information will be updated with the relevant information.
