Restraints - OC3 Mat Cannot Be Enabled

SI B 72 14 06
Safety Restraint Systems

October 2006
Technical Service

This Service Information bulletin supersedes S.M. B72 201 05 dated January 2005.

SUBJECT
Tips on Installing and Enabling OC3 seat occupancy mat

MODEL
E65, E66 (7 Series), E60 (5 Series), E83 (X3), E85 (Z4)

SITUATION
After replacing seat mat, the replacement mat cannot be enabled.

CAUSE
Possible causes are: Incorrect date in DISPlus/GT1, incorrect date input during the enabling process, tension in seat cover, weight on seat, excessive stress during installation.

PROCEDURE
Disconnect the battery. Follow repair instructions for replacing the seat cover (refer to RA65 77 600). Reconnect the battery. Before starting the enabling procedure, check the date and time in DISPlus/GT1 and correct if necessary.
1. With the new seat mat installed in the vehicle, eliminate any tension in the seat mat caused by the installation process by having an adult sit in the seat.
2. Remove all weight from the seat.
3. Using DISPlus/GT1, select Complete Vehicle/Service Functions/Seat Occupancy Detection/Seat Occupancy Mat, and then select Test Plan.
4. Follow the screen prompts to enable the seat mat.

- Note 1: When requested for the dealer number, select option 2 (default dealer number)

- Note 2: When requested for the date, be sure to input in the format shown on the screen.

If the mat fails to enable, regardless of the screen message, recheck the date inputted against the date in the DISPlus/GT1 (they both must be the same), be sure there is no weight on the seat, and any tension on the seat cover has been relieved.

Select Control Unit Functions/Read Fault Memory/Clear Fault Memory (to clear unrelated MRS faults).

Repeat process from step 3 above.

If necessary, perform a battery reset and repeat.

Clear any remaining fault codes.
