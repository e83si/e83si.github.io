Output Speed Sensors




Mechatronic Speed Sensors For Turbine And Output

Functional description, see BMW Technology:
TIS -> Document -> SI Technology -> SBT Number
- 2401 03 040

Fault detection
In the event of a fault, the following faults are stored:
- Turbine speed sensor: 4EE8
- Output speed sensor: 4EE9
- Engine-speed sensor, output - gradient too high: 4EEB
