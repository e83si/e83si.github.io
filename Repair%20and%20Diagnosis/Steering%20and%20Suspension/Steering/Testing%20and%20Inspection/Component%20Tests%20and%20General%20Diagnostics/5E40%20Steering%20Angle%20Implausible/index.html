5E40 Steering Angle Implausible



5E40 Steering Angle Implausible

Possible cause(s) of fault:
- Faulty calibration
- Vehicle track misadjusted
- Incorrectly coded
- The steering angle sensor can be twisted on the steering column
- The steering angle sensor is not supplied via terminal 30

Storage condition(s):
- The DSC continuously checks internally whether the values measured by the sensors are plausible. If the steering angle does not match up to the other values, the fault is entered in the DSC.
- If the steering angle sensor has no power supply via terminal 30, it must determine the current steering wheel position via parameters after each ignition on. If this fails, the fault entry is made in the DSC.

Troubleshooting procedure:
- Read out the fault memory of the steering angle sensor If the causes of the fault mentioned above are present, a fault must also have been entered in the steering angle sensor.
- Exception: interruption of the power supply (terminal 30): Here, a certain number of interruptions are necessary until a fault is entered in the steering angle sensor. Check the power supply (terminal 30) of the steering angle sensor In the test module B3714_00001 under (steering angle sensor/supply) there is an interruption counter for terminal 30.

NOTE: Wire tests and checks for loose contacts are possible via "Preset measurements".
