Multifunction Steering Wheel, 2nd Generation



MUltifunction Steering Wheel, 2nd Generation

The driver can use the multifunction steering wheel (MFL) to operate various vehicle functions from the steering wheel, depending on the equipment fitted. These functions include:
- Radio functions
- Telephone functions
- Cruise control functions
- Recirculated air control functions
- Steering wheel heater

In addition to a high degree of comfort, the multifunction steering wheel also offers the driver increased safety as it is not necessary to take the hands off the steering wheel to operate various functions. The driver can implement the functions from the multifunction steering wheel without diverting attention from the traffic situation.

The response times to MFL functions are so fast that no in feedback is noticeable.

The multifunction steering wheel (MFL) consists of the following components:
- MFL operating unit: Switch blocks for the MFL functions are provided on the steering wheel on the right and left next to the airbag unit.
- MFL control module: The MFL control module is integrated in the right switch block. The radio and telephone functions integrated in the MFL operating unit are transferred via the K-bus to the corresponding control modules. The button signals for cruise control are sent via a separate link to the cruise control module (Tempomat).
- Contact unit: The voltage supply and K-bus link are connected by means of a coil spring to the MFL control module.

Variants
Different versions of the multifunction steering wheel (MFL) are offered. The following section describes the functions of the maximum version.

Cruise control operation
A switch block for cruise control operation is located on the right side of the steering wheel. Description of operating keys from top to bottom:
- Resume: The vehicle accelerates or decelerates to the driving speed last set and maintains this speed.
- Accelerate: The speed is increased by 1 km/h by briefly hitting this button. If pressed longer, the speed is increased until the button is released again.
- Decelerate: The speed is decreased by 1 km/h by briefly hitting this button. If pressed longer, the speed is decreased until the button is released again.
- Off: Switches off the cruise control function.

Radio control
A switch block for radio operation is located on the left-hand side of the steering wheel. Description of operating keys from top to bottom:
- Search upward: With radio mode: Upward radio station search function. Pressed once during cassette mode: Forward track search. Pressed twice during cassette mode: Fast forward to end of tape. If this key is pressed during fast forward, the cassette drive switches over to PLAY.
- Volume + : Increases volume as long as button is pressed.
- Volume - : Lowers volume as long as button is pressed.
- Search upward: With radio mode: Downward radio station search function. Pressed once during cassette mode: Backward track search. Pressed twice during cassette mode: Rewind to start of tape. If this key is pressed during rewind, the cassette drive switches over to PLAY.
- Radio / telephone change over: This button is used to switch the search function between radio and telephone. The radio display (frequency or station) or the telephone display (phone number or name) is shown in the radio.

Telephone operation
A switch block for telephone operation is located on the left-hand side of the steering wheel. Description of operating keys from top to bottom:
- Search upward: Selection of the next telephone display (phone number or name) with display in the radio.
- Volume + : Increases volume as long as button is pressed. Only possible if "hands-free" is switched on.
- Volume - : Lowers volume as long as button is pressed. Only possible if "hands-free" is switched on.
- Search downward: Selection of the previous telephone display (phone number or name) with display in the radio.
- Radio / telephone change over: This button is used to switch the search function between radio and telephone. The radio display (frequency or station) or the telephone display (phone number or name) is shown in the radio.
- Hands-free: Dials the phone number displayed in the radio.

Recirculated air operation
The key for the recirculated air function is located on the right switch block. The recirculated air function can be alternately switched on and off.

The recirculated air control button is not fitted if a steering wheel heater is installed.

Steering wheel heater
The steering wheel heater is ready for operation as of terminal 15. It is switched on by pressing the key once on the multifunction steering wheel. The steering wheel heater is switched off again by pressing the key a second time.
