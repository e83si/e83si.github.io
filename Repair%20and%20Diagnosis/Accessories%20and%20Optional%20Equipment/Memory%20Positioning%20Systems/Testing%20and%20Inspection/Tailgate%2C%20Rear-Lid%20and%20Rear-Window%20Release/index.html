Tailgate, Rear-Lid And Rear-Window Release

The tailgate/trunk lid/rear window release makes it possible to unlock the tailgate/trunk lid (on Sedan, Coup� and Touring) as well as the rear window (on Touring) electrically.

Unlocking is by means of an electrical drive with integrated return spring.

Trunk lid release

Function
The drive is activated directly by the general module 5.

On E46 Sedans and Coup�s, it will only be possible to unlock the rear lid if the rear-lid lock is not in the workshop or hotel setting (lock cylinder horizontal) and the vehicle is stationary. The workshop or hotel setting is recognized by the general module via the VRHK line.

The rear lid/tailgate can be unlocked:
- Via the rear lid push button when the vehicle is unlocked
- Via the button in the vehicle interior (optional extra), also with locked vehicle
- Via remote control, also when the vehicle is locked

NOTE: Remote control functions are executed only if there is no key inserted in the ignition lock.

Emergency operation
The rear lid can be opened mechanically if the electric unlocking drive malfunctions.

On the E46 Sedan and Coup� via the luggage compartment lock (luggage compartment lid lock barrel).

On E46 Tourings via a mechanical emergency actuator inside the car. The emergency actuator is in the luggage compartment on the right side wall behind the side trim panel.

Rear-window release (Touring)

Function
The rear-window unlocking enables electrical unlocking of the rear window on "Touring" vehicles. The drive is activated by means of a separate relay activated by the general module.

It is only possible to release the rear lid/tailgate when the vehicle is stationary.

The rear window is released by pressing a push button (switch S172) in the tailgate.
