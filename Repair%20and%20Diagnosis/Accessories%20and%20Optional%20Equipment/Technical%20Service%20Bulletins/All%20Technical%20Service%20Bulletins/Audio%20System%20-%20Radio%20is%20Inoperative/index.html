Audio System - Radio is Inoperative

SI B 65 41 06
Sound System, Cruise, Alarm

December 2006
Technical Service

SUBJECT
Radio is Inoperative on New Vehicles

MODEL
E83 (X3)
E85 (Z4)
E86 (Z4 Coupe)

Vehicles produced from 09/2006.

SITUATION
Radio is inoperative on new vehicles after deactivating the transport mode.

CAUSE
Incorrect DIS version used to deactivate the transport mode.

INFORMATION
Do not replace parts!

Install DIS V47 and deactivate the transport mode again.

Note:
To avoid the additional work make sure that DIS V47 is installed on the DISplus or GT1 prior to deactivating the transport mode.

Older DIS versions will not work with the vehicles produced from 09/2006 on.
