Audio System - No Audio From Retrofitted CD Changer

SI B65 01 06
Audio, Navigation, Monitors, Alarms, SRS

February 2012
Technical Service

This Service Information bulletin supersedes SI 65 01 06 dated March 2006.
[NEW] designates changes to this revision

SUBJECT
Retrofitted CD Changer Has No Audio Output

MODEL
E83 (X3) from September 2005
E85 (Z4 Roadster) from January 2006

Vehicles with option 662 (Business CD Radio)

SITUATION
There is no audio from the retrofitted CD Changer, even when the CD # and CD track are shown in the display.

CAUSE

1. The "old generation" CD Changer is not compatible for these vehicles.

2. The "new generation" CD Changer is not retrofitted (coded) to the vehicle.

CORRECTION

1. Install a "new generation" CD Changer.

2. [NEW] Retrofit the CD Changer via ISTA/P to adapt option 672 to the vehicle.

[NEW] PROCEDURE

Install the CD Changer. Refer to the installation instructions on ASAP for the specific vehicles. After installing the CD Changer, the following retrofit must be performed in ISTA/P:

^ Retrofit, CD changer

Note that ISTA/P will automatically reprogram and code all programmable control modules that do not have the latest software.

For information on programming and coding with ISTA/P, refer to CenterNet / Aftersales Portal / Service / Workshop Technology / Vehicle Programming.
