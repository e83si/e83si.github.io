Navigation System - Missing or Incorrect Road Map Data

SI B 84 04 05
Phone, Navigation Systems

January 2007
Technical Service

This Service Information bulletin supersedes S.I. B84 04 05 dated April 2005.

[NEW] designates changes to this revision

SUBJECT
Navigation System Has Missing Or Incorrect DVD/CD Road Map Data

MODEL
Vehicles equipped with Navigation System (option609).

E39, E60, NEW E61 (5 Series)
E46, E90, NEW E91, NEW E92 (3 Series)
E52 (Z8)
E53, NEW E70 (X5)
E63, E64 (6 Series)
E38, E65, E66 (7 Series)
E83 (X3)
E85, [NEW] E86 (Z4)

SITUATION
The Navigation system route guidance is incorrect, navigation is not possible to certain addresses or road data is missing.

CAUSE
The NAVTEQ(R) DVD/CD Road Map has incorrect or missing data.

INFORMATION
Do not replace parts!

It is not a problem in the BMW Navigation systems. The Road Map disc data is the sole responsibility of NAVTEQ(R).

Inform the customer of the NAVTEQ(R) end-user license agreement which comes with the DVD/CD Road Map. (see attachments)

[NEW] The customer must report the error directly to NAVTEQ(R) .

[NEW]IMPORTANT:
- The customer must have the latest available NAVTEQ(R) Road Map before reporting an error. Road Map Error Reports on older Road Maps will not be processed by NAVTEQ(R).
- A valid email address must be provided so that NAVTEQ can reply to the customer feedback.

Additional information can be found on the NAVTEQ(R).

The NAVTEQ(R) web site has a section for reporting map errors (see attachments).
