Emergency Call (Without TCU 1.5 Telematic Control Unit 1.5)




ISIS emergency call
The following are possible by telephone
- an automatic emergency call by means of crash signal,
- a manually triggered emergency call via emergency call button.

For reasons of safety, a double data transfer between the SIM (Safety Information Module) and the telephone is installed.
- One data path follows the route SIM -> Byteflight data bus -> ZGM (central gateway module) -> System K-CAN data bus -> Control Display -> MOST data bus -> Telephone.
- In addition, there is a direct, serial, single-wire connection between the SIM and telephone. If the serial connection has a fault or fails, the path via the above bus links is used to trigger the emergency call.

After the emergency call has been triggered,
- an SMS signal is sent to the service provider: in cycles, the navigation system provides the telephone with a ready-made SMS (Short Message Service);
- a voice connection is set up.

Two numbers are set for the two data paths; these are dialed alternately until a connection is set up.

If no connection has been set up after 60 seconds, the user device dials the emergency number in accordance with the GSM standard (Global Standard for Mobile Telephone).

To provide support for any rescue operations that might be required,
- the interior lights are switched on (by the light module),
- the hazard warning system is switched on (by the light module),
- the central locking system is unlocked (via the Car Access System).

Brief description of components

Telephone and conditions for the emergency call
The following factors are decisive for the operational capability of the emergency call:
- The telephone and the telephone interface, including antenna system, are ready for service.
- There is an adequate power supply.
- The telephone card is in the card reader or: the emergency SIM is inserted (Subscriber Identity Module, i.e. the key card for the telephone).
- The connection between the telephone and service provider has been set up or can be set up.

Emergency call button
Pressing the emergency call button triggers the emergency call. The emergency call button is fitted in the center above the windshield in the roof lining. The emergency call button has diagnostic capability: the telephone interface can detect whether the button is connected.

If a fault occurs, a fault code is stored in the fault memory. In addition, a signal can be output (Control Display or voice output).

control modules
The following control modules are involved in the message transmission for the emergency call:
- SIM (Safety Information Module)
- ZGM (Central Gateway Module)
- LM (Light Module): the light module controls and monitors the vehicle lights. Information is received and sent via the data bus K-CAN SYSTEM. Among other things, the light module also controls the hazard warning system and the interior lights.
- CAS (Car Access System): The Car Access System implements the functions anti-theft alarm, radio remote control, ignition lock, terminal control, immobilizer, and central locking system. The CAS control module is also the gateway module between the System K-CAN and Peripherals K-CAN data busses.
- CD (Control Display): The Control Display in the center of the instrument panel indicates that an emergency call has been triggered. The Control Display indicates the current location of the vehicle as well as a confirmation when the connection to the emergency call center has been set up.
- NAVI (Navigation system): in the event of an emergency call, the navigation system outputs the current location of the vehicle.
- FBI (Flexible Bus Interface): In the Japan country version, the FBI transfers the emergency call signal to the telephone.

Main functions

Automatic emergency call by means of crash signal
If the ISIS system detects a crash of sufficient severity, the emergency call by telephone is triggered automatically. The emergency call contains the following data:
- Current position of the vehicle (from NAVI)
- Current vehicle data: vehicle speed at accident, accident severity, number of triggered airbags

The Control Display indicates a confirmation when the connection to the emergency call center has been set up.
- Confirmation that the emergency call has reached the emergency call center and
- confirmation that a voice connection has been set up via the hands-free facility.

Once an emergency call has been triggered, it cannot be cancelled.

Manual emergency call via emergency call button
In an emergency, the driver or passenger presses the emergency call button. The current vehicle position is then displayed in the Control Display along with a message indicating that the emergency call has been placed.

The Control Display indicates a confirmation when the connection to the emergency call center has been set up.
- Confirmation that the emergency call has reached the emergency call center and
- confirmation that a voice connection has been set up via the hands-free facility.

Once an emergency call has been triggered, it cannot be cancelled.

Interior lights
In the case of crash detection, the interior lights are switched on automatically by the light module. The interior lights can be switched off again
- using the button for the interior lights,
- by switching off the ignition and switching on again.

Hazard warning system
In the case of crash detection, the hazard warning system is switched on automatically by the light module.

Central locking
In the case of crash detection, the central locking system is automatically unlocked by the CAS (Car Access System).

Service functions

Note on safety
When working on the roof lining or on the button for the sunroof, do not inadvertently trigger an emergency call manually (emergency call button)!

Country-specific version

Japan: FBI Flexible Bus Interface
The two data paths for the emergency call are
- SIM -> Byteflight data bus -> ZGM -> System K-CAN data bus -> Control Display -> MOST data bus -> FBI -> serial cable -> Telephone and
- the additional direct, serial, single-wire connection between the SIM and FBI. There is also a serial line from the FBI to the telephone.

If the automatic emergency call is triggered, the Japan telephone sets up emergency call voice communication with the police. The telephone (or FBI) ignores the accident severity data sent by the SIM, as no emergency call service based on data is in place in Japan.
