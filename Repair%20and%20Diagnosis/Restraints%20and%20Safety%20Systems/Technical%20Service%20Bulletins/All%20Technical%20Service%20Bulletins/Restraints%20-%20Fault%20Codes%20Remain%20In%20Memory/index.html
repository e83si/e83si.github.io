Restraints - Fault Codes Remain In Memory

SI B 65 28 09
Audio, Navigation, Monitors, Alarms, SRS

September 2009
Technical Service

SUBJECT
Not Possible to Delete Faults in the MRS5K Using Quick Delete

MODEL
E83 (X3)

SITUATION

Following a quick delete in ISTA, fault codes for the MRS5K (Multiple Restraint System) remain in the fault memory list.

CAUSE

ISTA software error

PROCEDURE

1. Rectify the cause of the MRS fault in the vehicle.

2. Using the ISID, switch to the control unit tree diagram

3. Select the MRS control module

4. Select the "Call up CU functions" button

5. Select the "Component Actuation" tab

6. Select the "Clear fault code memory" menu item

7. Start the deletion process by selecting "Actuate Component"

8. Perform a vehicle test to confirm that the faults have been successfully deleted

WARRANTY INFORMATION

Information only
