Interior - One Touch Memory Function Information

SI B 52 08 10
Seats

August 2010
Technical Service

SUBJECT
Front Seats: One Touch memory Function

MODELS
All models

SITUATION
There may be some confusion as to the operation of the "one touch memory function", when requesting a saved seat position. In most cases the operation is normal. This service information bulletin will clarify the normal operation below.

INFORMATION

When using the seat memory, there are two options for requesting a successfully saved seat position from the vehicle, depending on the operating mode.

1. With the vehicle unlocked and terminal 15 off, press the desired memory button once, briefly. The seat moves to the stored position. This is considered the "one-touch function".

2. With the vehicle unlocked and terminal 15 on (ignition on or while driving): the desired memory button must be pressed and held until the seat moves to final stored position. Under all circumstances, avoid moving the seat to a position which is uncomfortable for the driver while the vehicle is in motion. This situation does not allow the "one-touch function".
