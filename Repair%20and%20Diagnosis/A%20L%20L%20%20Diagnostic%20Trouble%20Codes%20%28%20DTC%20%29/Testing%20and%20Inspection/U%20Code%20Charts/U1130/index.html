U1130

2F11 - U1130

Information saved in
DME

Fault code
2F11 - U1130

Fault text
Radiator blind, top

Fault description
Lost Communication With Radiator Shutter (Upper)

Condition for fault identification
Test condition:
1. AKKS must be initalized.
Operation of lower flap:
3 x open and 3 x closed
Or conduct the system test
Terminal condition: Terminal 15 ON
Voltage condition:
Voltage in onboard electrical system > 10 V

Condition for fault memory entry
none

Action in service
1. Check voltage supply to AKKS (focus especially on GND
[PIN 2] at the valve)
2. If OK, then replace AKKS

Fault effect and breakdown warning


Driver information
Warning light:
no external display;

Service instruction
none
